# Jakob Wiedmer – *Flut* (Kapitel I–XLI)
Digitale Lektürearbeit (offlinefähig, GitHub Pages geeignet)

## Dateien
- `index.html` – interaktive Lektürearbeit
- `JakobWiedmer_Flut_web.pdf` – weboptimiertes PDF (eingebettet)
- `wengen_blur.jpg` – geblurrter Hintergrund

## Funktionen
- Kapitelweise Freischaltung (Fortschritt wird gespeichert)
- Automatisches Speichern aller Eingaben (LocalStorage)
- Schülerinnen-Modus / Lehrerinnen-Modus (passwortgeschützt)
- Separates Lehrerinnen-Kommentarfeld pro Kapitel (nur im Lehrerinnen-Modus sichtbar)
- Fortschrittsanzeige (freigeschaltete Kapitel)
- Export als PDF über Druckfunktion (Button „Als PDF exportieren“)
- Offlinefähig (keine externen Abhängigkeiten)

## Lehrerinnen-Modus
- Umschalten oben rechts
- Passwort: **flut**
- Lehrpersonen sehen zusätzliche Kommentar-Felder pro Kapitel

## Datenschutz
- Es findet **keine** Datenübertragung statt.
- Alles wird lokal im Browser gespeichert (LocalStorage).

## GitHub Pages
1. Dateien ins Repo-Root hochladen.
2. Settings → Pages → Deploy from branch → `main` / root.
3. Seite aufrufen: es funktioniert direkt.

## Tipp
Für eine kapitelweise PDF-Ausgabe: nur gewünschte Kapitel freischalten → „Als PDF exportieren“.
