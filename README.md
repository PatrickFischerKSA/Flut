
# Jakob Wiedmer – Flut
Digitale Lektürearbeit (Kapitel I–XLI)

## Kurzbeschreibung
Diese Webanwendung dient der strukturierten Lektürearbeit zu Jakob Wiedmers Roman *Flut*.
Sie ist für den Einsatz im Unterricht (Sek II) konzipiert.

## Funktionen
- Kapitelweise Freischaltung
- Automatische Speicherung (LocalStorage)
- Schülerinnen- / Lehrerinnen-Modus (passwortgeschützt)
- Separates Lehrerinnen-Kommentarfeld
- Fortschrittsanzeige
- PDF-Export
- Eingebettetes weboptimiertes PDF

## Lehrerinnen-Modus
Passwort: **flut**

## Technik
- Offlinefähig
- GitHub-Pages-tauglich
- Keine externen Abhängigkeiten
